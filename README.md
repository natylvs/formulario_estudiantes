# formulario_estudiante

El proyecto consiste en el desarrollo de un formulario que permite calcular el promedio de un estudiante a partir del registro de cuatro calificaciones principales. El sistema procesa automáticamente estas notas y determina si el estudiante ha aprobado o reprobado según el resultado obtenido.

Si el promedio inicial no alcanza la calificación mínima requerida, el programa habilita un campo adicional para ingresar la nota del completivo y realiza nuevamente el cálculo. En caso de que el resultado continúe siendo insuficiente, se activa una última casilla para registrar la nota del extraordinario, mostrando finalmente el promedio definitivo y el estado académico del estudiante.

Explicación de uso del programa

El usuario debe ingresar las calificaciones correspondientes en los campos Nota 1, Nota 2, Nota 3 y Nota 4. Una vez completados todos los valores, debe presionar el botón Calcular, y el sistema procesará automáticamente la información ingresada para obtener el promedio del estudiante, el cual se mostrará en el campo correspondiente.

Si el promedio obtenido es superior a 69, el programa indicará que el estudiante está Aprobado y no habilitará campos adicionales. En caso de que el promedio sea igual o inferior a 69, se activará el campo Completivo para ingresar una nota extra. Si al incluir esta calificación el promedio alcanza 70 o más, el estudiante será aprobado y no será necesario utilizar la nota extraordinaria.
